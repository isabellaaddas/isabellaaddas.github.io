Assignments

  • Assignment 1 - Basic HTML

    This assignment demonstrates my ability to write a simple web page in HTML. I used images and wrote text about the band GHOST to prove my understanding of the language with key elements like DOCTYPE, title, etc.

  • Assignment 2 - Basic CSS

    This assignment demonstrates my understanding and knowledge of CSS in creating a simple web page. The page itself is built in HTML and uses a stylesheet to add a div, style a proper table, and implement a color scheme.

  • Assignment 3 - Page Layout

    This assignment demonstrates my ability to use a flexbox correctly in tandem with other HTML5 elements. I utilize several elements like articles, sections, asides, etc. with flex styling to match on large and small screens.

  • Assignment 4 - Recreate CSS

    This assignment demonstrates my knowledge of CSS and flex by being able to recreate any web page given to me. I use many techniques in CSS such as ::after, flex, relative and absolute positions, etc.

  • Assignment 5 - JavaScript Intro

    This assignment shows my newfound knowledge of JavaScript, specifically in using arrow functions, constants, and more. I use a script file to define functions for specific events like clicking page elements.

  • Assignment 6 - JavaScript Conditionals

    This assignment displays my knowledge of JavaScript conditionals by utilizing intervals, the Date class, sliders, and more to display differing information based on each condition.

  • Assignment 7 - JavaScript Loops

    This assignment demonstrates looping knowledge by drawing a simple scene with CSS art on the home page. I use a for loop to insert every object onto the screen and some conditionals for time of day.

  • Assignment 8 - JavaScript Arrays

    This assignment utilizes my knowledge of arrays in the JavaScript language to create associative arrays for each cat and their pictures. I use a for loop to set up the page and use click events to display the popup.

  • Assignment 9 - JavaScript Classes

    This assignment displays my ability to create classes in JavaScript and using them to define objects with certain properties, displayed in HTML elements. Every section is made with these classes and their information.
